# V-Disk Explorer

A futuristic file explorer web app with a mechanical hard drive aesthetic. Features file management, drag-and-drop uploads, and a retro computing vibe.

![V-Disk Explorer](https://img.shields.io/badge/version-1.0.0-orange)
![React](https://img.shields.io/badge/React-19-blue)
![Vite](https://img.shields.io/badge/Vite-6-purple)

## Features

- 🖥️ **Retro Hardware Aesthetic** - Mechanical hard drive inspired UI design
- 📁 **File Management** - Create folders, upload files, organize your virtual storage
- 🎨 **Theme Support** - Dark/Light mode toggle
- 💾 **Local Storage** - All data stored locally in your browser
- 🎯 **Drag & Drop** - Drag files directly into the app to upload
- 🔍 **Search & Filter** - Find files by name, type, size, or date
- 📱 **Responsive Design** - Works on desktop and tablet devices

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/v-disk-explorer.git
cd v-disk-explorer

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

The built files will be in the `dist/` folder.

## Deployment

### Vercel (Recommended)

1. Go to [Vercel Dashboard](https://vercel.com/dashboard)
2. Click "Add New → Project"
3. Import your GitHub repository
4. Vercel auto-detects the configuration
5. Click "Deploy"

### Netlify

1. Go to [Netlify Drop](https://app.netlify.com/drop)
2. Drag and drop the `dist` folder
3. Get your live URL

### Manual Deployment

Upload the contents of the `dist` folder to any static hosting service.
