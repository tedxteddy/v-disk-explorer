## V Disk Explorer - The App

This is an open source virtual hard disk explorer web application with file management, demo chat, and customizable settings.

### Features

- **Virtual File Explorer** - Browse and manage virtual files and folders
- **Demo Chat** - Local messaging with auto-reply simulation
- **Settings** - Privacy controls and demo toggles
- **Dark/Light Theme** - Toggle between modes
- **Local Storage** - All data stored locally

### How to Use

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
```

### Deploy

Deploy to Vercel, Netlify, or any hosting service.
