# V Disk Explorer

A virtual hard disk explorer web application with file management, demo chat, and customizable settings.

## Features

- **Virtual File Explorer** - Browse and manage virtual files and folders
- **Demo Chat** - Local messaging with auto-reply simulation (no backend required)
- **Settings** - Privacy controls and demo toggles
- **Dark/Light Theme** - Toggle between dark and light modes
- **Local Storage** - All data stored locally in your browser

## How to Use

### Run Locally

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up environment:**
   Copy `.env.example` to `.env.local` and add your `GEMINI_API_KEY` (optional - for AI chat features)

3. **Run the app:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

### Deploy

This project can be deployed to any hosting service like Vercel, Netlify, or render.
